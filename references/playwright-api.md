# Playwright MCP API Reference

Playwright MCP controls a headless Chromium browser instance for visual testing.

## Tools
- `playwright/browser_navigate`: Navigate to a URL.
- `playwright/browser_click`: Click on a CSS selector.
- `playwright/browser_type`: Type text into an input selector.
- `playwright/browser_take_screenshot`: Capture page screen.
- `playwright/browser_tabs`: List active browser tabs.
- `playwright/browser_evaluate`: Evaluate Javascript in browser page.
- `playwright/browser_network_requests`: Capture network logs.
